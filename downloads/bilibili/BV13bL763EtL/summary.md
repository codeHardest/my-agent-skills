### 二、报告核心结论

#### 1. GDP增速预测
- 2026年：上调0.1个百分点至**4.8%**
- 2027年：上调0.1个百分点至**4.7%**
- GDP平减指数：上调0.3个百分点至**0.5%**

#### 2. 双速经济（Two Speed Economy）概念
**定义**：即便经济总体增速稳定，但内部分化严重，外需与内需以两个完全不同的速度增长。

**外需强劲的原因**：
- 中国完备的产业链，相较其他经济体受中东能源冲击更小
- 中东冲突后，各国对"新三样"（新能源产品）需求增长
- 中国在全球出口份额仍有上升空间

**内需疲软的原因**：
- 居民消费增速放缓
- 投资更多是行政主导下的增长

### 三、出口分析

#### 核心数据
- 2026年出口名义增速：**10%**（双位数增长）
- 进出口对实际GDP增长贡献：**1.3个百分点**

#### 短期判断
大摩认为短期中国出口虽然可能受到海外油价冲击导致贸易伙伴需求萎缩影响，但：
- 海外供给也受到负面冲击（产业链不完备）
- 中国出口份额将提升4-5个百分点
- 出口份额提升可抵消需求萎缩冲击
- **总量下降，但中国占比提升，两者相抵影响不大**

PMI数据印证：4月新出口订单回到扩张区间50.3，创近几年最高。

#### 中长期驱动
1. **AI超级周期（Super cycle）**：对半导体等电子设备需求持续增长，中国在中低端半导体生产有优势
2. **能源转型**：中东冲突后各国效仿中国建设可再生能源和储能设施，"新三样"将迎来新爆发

#### 专栏一：为何出口高增长无法转化为就业
**核心问题**：为何出口高歌猛进但就业市场无好转？

**四大原因**：

1. **房地产仍是拖累**：要抵消房地产拖累，需要出口名义增速维持14-15%高位，目前远未达到

2. **出口商品资本密集型化**：
   - 出口产品向价值链上游转移：电动汽车、油轮、半导体、能源设施
   - 均为资本密集型、自动化工厂生产
   - 每亿美元出口对应产生的新就业量远低于历史水平

3. **产能过剩**：
   - 2022-2024年大量制造业投资导致产能过剩
   - 企业拿到出口订单后，提高产能利用率即可完成，无需新建厂房产线
   - 新建需求下降进一步削弱就业拉动

4. **高利润行业有限**：
   - 出口真正转化为工资增长的领域局限在少数细分行业
   - 未见广泛意义上的收入增长和就业增长

**数据图示**（图三）：
- 横轴：企业收入增速
- 纵轴：就业增长
- 2018年对比2013年：斜率陡峭（收入增长→就业增长）
- 2023年对比2018年：斜率下降（收入增长但不带动就业）

### 四、内需分析

#### 专栏二：AI应用如何拖累就业复苏

**长期**：AI对经济有上行支撑
- 未来10年全要素生产率将上行3个百分点
- 到2035年累计GDP可多增长3.5个百分点

**短期路径颠簸曲折**：
- 企业盈利困难，更倾向于用AI替代员工成本而非提升员工产出
- AI加速替代劳动岗位，而非创造新岗位

**AI替代的三个层次**：
1. 生成式AI：替代白领工作
2. AI Agent：替代中级专业岗位
3. 物理AI（机器人、无人机）：替代劳动密集型服务业岗位

**负反馈循环**：
```
AI替代 → 降薪裁员 → 通缩加剧 → 企业亏损 → 无法继续AI投资 → AI替代加速
```

#### 消费预测
- 2026年消费增速：**3.7%**
- 较2025年的4.5%下滑0.8个百分点
- GDP增速贡献相应下滑

**消费不振原因**：
- 就业压力大，官方失业率3月达**5.4%**
- 25-29岁青年失业率创历史新高**7.7%**
- 居民预防性储蓄未来5年维持高位
- 私营部门消费持续低迷
- 供给侧为主政策框架，对民生是兜底而非广泛支持

#### 投资预测
- 固投增速：2025年2% → 2026年**3.5%**（反弹1.5个百分点）

**三大驱动力**：
1. AI相关资本开支
2. 行政主导的基建投资（有钱有项目）
3. 出口驱动的部分制造业投资

**基建投资细节**：
- 以准财政工具推动
- 去年结余5000亿+新资金足以抵消土地出让金下滑
- 去年是"十四五"收官年，受双重约束（钱不够+无好项目）；今年新项目上马

**房地产**：
- 投资仍低迷
- 大摩坚持：预计一线二线城市房价**2027年**才能广泛企稳（唯一坚定此观点的知名外资投行）
- 企稳后各项指标仍将保持低迷

### 五、物价与再通胀

#### 当前再通胀不可持续的原因
- 当前是**输入性通胀**
- 内需疲软和劳动力市场疲软导致下游价格难以上行
- 不伴随居民名义收入增长
- CPI相对平稳，PPI近两月大幅上行

#### 预测数据
- GDP平减指数上调至0.5%，但远低于CPI和PPI涨幅
- 2027年PPI将转为**-0.4%**（负值）

#### 专栏三：日本经验借鉴

**日本走出通缩路径**：
- 2020年疫情后各国工厂停产、刺激政策导致物价反弹
- 大多数国家随后加息抗通胀
- **日本选择**：维持长期低利率，导致日元大幅贬值
- 日本政府提出"Comprehensive Economic Package For Complete Exit From Deflation"（完全退出通缩的综合经济方案）
- 财政政策配合：税费减免、商品补贴、支持本土投资、居民家庭定向支持

**传导机制**：
- 2023年Q1：物价向工资传导开始加速
- 2023年Q3：工资向物价传导形成正向反馈
- 2024年3月关键确认点：大型企业协商工资反映当年名义工资增速**5.85%**，创31年最高

**核心启示**：
- 输入性通胀只是再通胀的**催化剂（Catalyst）**
- 需要广泛且持续的逆周期政策（货币+财政）配合
- 真正的再通胀（True Reflation）：名义GDP增速跑赢通胀，名义收入增速跑赢通胀，实际收入增速增长

**中国的现实**：
- 决策层对逆周期政策"嗤之以鼻"，倾向"托而不举"（Cushioning not Lifting）
- 出口表现强劲，市场对政策无预期
- 大摩放弃降息预期，认为2026年全年无降息
- 有限财政空间仍偏好供给侧（科技自立自强、能源安全）
- 对消费支持是兜底而非广泛支持
- 地缘政治冲突加强了对能源战略安全和科技自立自强的重视

### 六、汇率展望

#### 预测数据
- 2026年末：美元兑人民币 **6.75**
- 2027年末：美元兑人民币 **6.8**

#### 关键逻辑
- 人民币汇率相对稳定，变化主要取决于美元走势
- 人民币相较一篮子货币有**3-4个百分点**升值空间
- 美元看法：2026年持续贬值（全球外汇团队预测）
- 2027年上半年美国经济增速可能再次提升，美元可能走强

### 七、风险提示

#### 上行情景（Upside Scenario）
- 外需超预期强劲
- 决策层对国内消费提供更多支持

#### 下行情景（Downside Scenario）
- 油价最终导致全球衰退
