1. 该行业在2025年的利润增速排名倒数

3. 在此基础上对比2026年一季度的经营情况
4. 如果一季度利润增速又开始明显复苏的也会被排除

### 衰落行业第五名：医疗美容

**核心数据**：
- 2025年：A股上市的医疗美容公司合计利润下滑28.76%
- 2026年一季度：继续下行57.3%

**行业历史回顾**：
- 2019年到2022年，"医美三剑客"华西生物、爱美客和浩海申科相继在A股上市
- 依托超过80%的毛利率，一度成为堪比白酒的白马股明星
- 医美龙头公司爱美客被称为"医美茅"，股价最高涨到1300元，市盈率一度接近400倍
- 市场最流行的口号："男人的茅台，女人的医美"

**衰落原因分析**：
1. **市场竞争加剧**：2020年后，几乎所有医药公司都想进入医美行业分一杯羹，竞争从蓝海变为红海
2. **宏观消费环境恶化**：2021年后我国消费市场进入通缩螺旋，房地产崩溃导致高端白酒崩溃，医美更难以支撑
3. **价格战白热化**：玻尿酸降价30%，瘦脸针买一送一，高利润率神话破灭
4. **行业增速骤降**：从之前动辄60%-70%下滑到如今的10%
5. **产品持续内卷**：2025年新获批的52张三类医疗器械注册证中，玻尿酸类产品占25张

**价格崩塌数据**：
- 玻尿酸：从当年动辄上千元降价到了199元
- 童颜针：从最初近2万元降到了线上只要2999元

**代表企业现状**：
- 爱美客：股价从最高点下跌80%，2025年净利润同比下降34.05%，2026年一季度继续下滑32.79%
- 某互联网医美第一股：2025年全年直接亏损2.42亿元

**结论**：医美这个曾经被大家视为印钞机的行业，如今已经跌落神坛。

### 衰落行业第四名：橡胶

**核心数据**：
- 2025年：橡胶行业上市公司净利润增速为负的43.59%
- 2026年一季度：继续下滑至负的50.77%

**行业定义**：主要指中下游的橡胶加工企业

**供给端分析**：
- 2016年后上一轮橡胶行业种植扩产潮结束
- 橡胶树从种植到产胶需要6-7年，橡胶原料的供给增量枯竭在2023年才开始显现
- 泰国：新增种植面积开始负增长，老一代橡胶树开始老化
- 印尼：因橡胶价格长期低迷，大面积改种每公顷收益是橡胶2.8倍的棕榈树

**成本传导**：橡胶原料价格从2023年后一涨再涨

**前期支撑因素**（2025年之前）：
- 合成橡胶因原油价格一直处于低位，价格受到压制
- 新能源汽车快速铺开
- 国家对汽车置换大力补贴
- 轮胎需求一直很不错
- 中国轮胎品牌全球市场占有率快速崛起

**2025年变天原因**：
1. **产能过剩**：2023年到2024年中国轮胎企业增加了至少1亿条轮胎的产能，2025年产能落地
2. **需求萎缩**：汽车置换大潮接近尾声
3. **成本暴涨**：中东冲突导致油价一路走高，带动合成橡胶暴涨
4. **需求下降通道**：我国汽车销量正式进入下降通道

**开工率数据**：
- 截止2025年11月下旬：半钢胎开工率跌破70%的盈利线
- 全钢胎开工率正在往60%狂奔

**三重暴击**：原材料大涨 + 新增产能集中落地 + 下游需求停滞

### 衰落行业第三名：房地产及其产业链

**连续三年上榜历史**：
- 2023年：利润下滑215.65%，登顶第一名
- 2024年：利润下滑2800.31%，蝉联第一
- 2025年：利润下行收窄至负的24.15%
- 2026年一季度：实现正增长0.4%，止跌

**当前困境**：
- TOP50上市房企2025年合计亏损1450亿
- 总亏损金额依旧是我国所有行业里的"扛把子"
- 超七成房企营业收入下滑，行业还在收缩
- 产业链（水泥、装修、建材、厨卫电器等）2025年依旧以负50%左右的利润下行速度负增长

**改善迹象**：
- TOP50上市房企总亏损金额相比2024年少亏1360亿
- 总负债下降至11.76万亿，相比2024年减少12.97%
- 行业衰落加速度阶段已经结束

**市场分化现象**：
- 新开发楼盘减少 + 新一代住宅出现
- 2026年开年后一线城市出现久违的量价齐升
- 上海放开内环核心区域后，新房和二手房价格持续上涨
- 但全国未形成燎原之势，5月后这波量价齐升逐渐失去声势

**未来预判**：
- 一线城市和强二线城市：企稳止跌
- 其他城市：继续向下探底
- 2026年下半年情况：明显差于上半年

**结论**：虽然衰落加速度阶段结束，但并不意味着行业开始复苏。

### 衰落行业第二名：生物制品

**核心数据**：
- 2025年：利润下滑111.07%
- 排名：去年第三，今年晋升第二

**行业定义**：医药一级行业下的二级行业，主要产品包括各种疫苗、诊断类试剂、血液制品等

**行业现状**：
- 2024年已陷入全行业亏损
- 2025年亏损进一步放大
- 体外诊断行业除几个龙头外，大部分企业利润跌到负值
- 多家企业被ST

**价格战惨烈程度**：
- 带状疱疹疫苗（仅两家公司竞争）：从每支千元以上直接跌破到500元
- 流感疫苗：一路杀到5.5元/支
- 智飞生物：曾经千亿市值的龙头，2025年亏损百亿，位列全体上市公司亏损第五名
- 其他疫苗企业几乎都在亏损线上下

**衰落根本原因**：
1. **疫情后遗症**：企业因疫情进入利润高光期，扩产产能疫情后成为内卷元凶
2. **集采压力**：医药集采全国常态化推进，范围持续扩大，降价幅度远超市场预期
3. **全方位挤压**：多种不利因素同时在近两年出现

**行业出路**：
- 出海化
- 差异化
- 创新化

**结论**：只有走这三条路，才能在寒冬里存活并实现第二春。留给各家企业的时间不多了。

### 衰落行业第一名：养殖

**核心数据**：
- **生猪和肉鸡养殖**（20多家上市企业）：
  - 2025年利润增速：负的45.84%
  - 2026年一季度：扩大至负的170.59%

- **水产养殖**（6家上市企业）：
  - 2025年利润增速：负的72.34%
  - 2026年一季度：暴跌至负的1209.46%

- **饲料**：
  - 2025年利润增速：负的54.24%
  - 2026年一季度：负的101.88%

**产品价格崩塌**：
- 鸡蛋：2025年跌破3块一斤，创出56年来新低
- 生猪：2026年最低跌到4块多一斤
- 养殖成本较高个体户：养一头猪倒亏500元
- 规模化养殖上市公司：全部进入亏损状态，几乎无一幸免

**水产养殖极端数据原因**：
- 中东冲突导致油价升高，传导到下游化肥
- 饲料价格伴随上涨
- 行业整体体量小
- 下游消费持续萎靡
- 产品价格持续下跌
- 业绩变脸更加夸张

**三大原因分析**：

**原因一：餐饮消费持续低迷**
- 居民部门在收入预期不佳之下持续降杠杆
- 有钱先还债，剩下的存银行
- 国家层面补贴只集中在汽车、家电等行业，吸收了本就不多的居民端流动性
- 其他消费行业受到双重挤压
- 严查公务宴请让高端酒店和餐馆客流量严重下滑
- 水产品餐饮采购量2025年下降30%

**原因二：进口牛羊肉价格竞争力强**
- 猪肉在我国消费占比持续下降
- 2018年：62.1%
- 2025年：57.9%

**原因三：企业逆势扩产（猪周期失效）**

猪周期逻辑：
1. 猪肉价格上涨 → 养殖户扩产
2. 大批生猪出栏 → 供大于求 → 肉价暴跌 → 养殖户亏损
3. 淘汰母猪 → 供应变少 → 肉价重新涨起
4. 周而复始，历史上约4年一轮

本次周期失效原因：
- **博弈心态**：大家都知道这个周期，肉价暴跌后很多人选择不减产，硬扛赌周期反转
- **效率提升**：头部企业养猪效率大幅提升，最低可以把成本做到5元/斤
- **成本线下移**：生猪成本线价格中枢持续下移

**结果**：
- 猪肉趴在底部多年，周期反转迟迟未出现
- 即使95%的企业都在亏损，生猪产能出清也迟迟不肯来
- 跌破历史极值的猪肉价格直接带崩了整个养殖行业
