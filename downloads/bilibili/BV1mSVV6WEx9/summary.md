## 详细内容（保留所有原始信息）

### 一、信号1：A股整体估值

**数据点**：
- 截止26年5月26日，万得全A（除金融、石油石化）适应率：45倍
- 分位点：88%
- 判断：已经比较高，但还没到极高程度

**历史规律分析**：
- 观察05年股权分置改革以后的几次牛市数据
- 07年大牛市、09年4万亿次级、15年杠杆牛市，顶点都在70倍适应率附近
- 考虑到投资越来越成熟，牛市泡沫应该越来越小
- 军师预测：下一次大牛市的顶点应在60倍适应率附近
- 26年5月的45倍适应率显然还有一定的上涨空间

**重要说明**：
- 60倍是总结历史规律之后的预测，仅供参考
- 前提假设：这次牛市是10年级别的大牛市
- 当前判断：这一轮牛市截止26年5月还不算全面大牛市，属于人工智能带动的结构性牛市
- 核心主线：算力相关的半导体芯片

### 二、信号2：半导体板块估值

**指数表现**：
- 中证全指半导体指数
- 2年（应为26年）4月初到5月底
- 不到两个月时间指数上涨超过60%

**估值数据（26年5月26日）**：
- 半导体的适应率（PE）：145倍
- 分位点：98%
- 当前位置已达到20年炒作半导体自主可控的高点
- 市净率（PB）：10倍
- 分位点：99%

**判断**：
- 从分位点看，当前半导体的估值已经是极贵
- 但这一轮AI带来的超强半导体周期有特殊性（以后再讲，今天不考虑特殊性）
- 结论：从估值看，中国A股整体还未到极贵程度，但主线半导体已极贵

### 三、信号3：新开户数

**历史规律**：大牛市见顶之前，新开户往往会先见顶

**数据点**：
- 26年1月新开户：492万（但很大程度受白银LOVE套利影响）
- 20年（应为26年）3月新开户：460万
- 4月开户数：249万

**分析**：
- 乍一看3月疑似见顶
- 但有季节性影响：每年春节后的3月份开户数量都会比较多
- 460万的绝对值也不算太高
- 军师判断：暂时不把它当做开户见顶信号

### 四、信号4：融资融券余额

**历史规律**：股市见顶时一般两融余额会很高

**数据点**（26年5月25日）：
- 两融余额：接近3万亿元
- 创了历史新高
- 乍一看很危险

**占比分析**：
- 两融余额占A股流通市值：约2.7%
- 处于正常偏高位置
- 军师个人标准：占比超过3%才值得警惕

**交易层面**：
- 历史规律：股市见顶之前，两融交易额占比一般会提前见顶
- 目前交易占比：约11%
- 也是正常偏高位置
- 没有达到需要警惕的位置

**说明**：这些数据的跟踪方法之前都讲过，不重复；后续会不定期通过文稿持续更新这些见顶信号

### 五、信号5：公募基金申购

**历史规律**：牛市见顶时，基金申购也会比较疯狂；牛市顶点一般有发行的巅峰

**数据点**：
- 26年4月和5月的份额增长：是负增长
- 看不到情绪的疯狂

**原因分析**：
1. 一是这一波行情不是所有股票整体涨，而是所有资金都往AI半导体流动
   - "万物落而AI生"
   - 存量资金此消彼涨
2. 二是这一波规模暴增的是私募量化基金
   - 私募基金不受跟踪
   - 公募的数字已经没有那么强的参考性了
   - 可惜私募量化没有特别可靠的数据

**结论**：基金申购这个信号参考意义已经不太大了

### 六、信号6：换手率与市场情绪

**换手率定义**：每天的交易量除以股票的总量
- 换手率越高 → 市场情绪越火热
- 换手率越高 → 市场分歧越大
- 当分歧大时总有一边是对的，市场很可能出现转向

**数据点**：
- 26年5月换手率并不高
- 没有体现出很强的炒股情绪

**定性判断**：
- 股市整体没有全民炒股的盛况
- 还没到"卖菜大妈都在聊股票"的时候
- 但市场针对AI半导体的情绪已经非常极致

**AI半导体的极端表现**：
- 上一轮消费医药之类的明星基金经理差不多都下课或者半下课
- 出现外资券商不再覆盖中国消费股的现象
- 所有资金都往AI上流动得特别快

**军师个人观点**：
- 这种情况本身就很危险
- AI（如英伟达）是不是泡沫我不敢说（独家垄断+生态壁垒）
- 是不是泡沫得看后面AI产业的发展
- 但**存储之类的很多电子元器件已经是泡沫了**：
  - 理论上可以无限扩产
  - 长期看没有供给瓶颈
  - 而且也不是垄断
- 不敢想象这轮超强半导体周期过去后会多么惨烈
- 目前只是大概的感觉，后续会做进一步研究分析

### 七、信号7：融资（IPO与再融资）

**逻辑**：
- 融资和减持一方面从市场抽取资金，一方面还增加股票
- 理论上对股市是利空
- 潜台词：公司股东认为股价比较贵了，能卖个好价钱
- 可作为观察牛市泡沫的指标

**当前情况**（26年5月）：
- 没看到太夸张的融资数据
- 无论是IPO还是再融资都不算大
- 历史上几次牛市顶部附近都有超大的IPO

**IPO预警**：
- 军师之前讲过：估值已经贵的时候，看到超大型IPO上市要警惕
- 之前推测这次很可能是长存（长江存储）、长鑫
- 26年5月底长鑫已经过会
- 顺利的话6月或7月就能上市
- 长江存储预计年内也能上市
- **这算是一个比较值得警惕的危险信号了**

### 八、信号8：减持

**数据可视化**：
- 绿色线代表净减持规模
- 24年9月股市大涨之前，重要股东净减持基本维持在零附近
- 之后随着股市上涨，减持规模逐渐增大
- 26年5月的减持：约400多亿（也不算多）

**警惕线**：
- 军师之前设置的警惕值：单月超过1000亿
- 还差得比较远

**结论**：
- 从减持数据来看没有出现危险信号
- 但这个统计是已经完成减持的数据，有一些滞后
- 最近减持公告明显多了起来
- 还有国家集成电路大基金减持的消息
- **这样来看也有一点危险的苗头了**

### 九、信号9：美股市场联动

**背景**：
- 中国A股早已不是独立的市场
- 军师之前讲过这一轮美股和A股很可能会同时见顶
- 美股26年以来半导体好似脱离地心引力一路向上

**费城半导体指数表现**：
- 25年（应为26年）4月到5月
- 两个月时间累计上涨80%
- 非常夸张

**与互联网泡沫对比**：
- 绿色线是1995-2001年互联网泡沫期间的费城半导体指数
- 把起点设为1，最高涨到接近5，即涨了4倍
- 纳斯达克综合指数涨幅差不多（为防图太乱未画）
- 如果把22年10月到26年5月这轮AI牛市的指数叠加上去
- **本轮牛市半导体指数已经涨了超过4倍**
- **最近两个月这种加速上涨的走势，跟互联网泡沫见顶之前的走势惊人相似**
- "预要使其灭亡，必先令其疯狂"
- 一般大牛市见顶之前都会有一段疯狂的时间
- **这段的疯狂算是一个危险信号了**

**特殊性说明**：
- 不是说互联网泡沫涨4倍AI也涨4倍就结束
- 这次AI有特殊性：
  - 之前互联网泡沫主要靠讲故事
  - 这轮AI一边讲故事，一边兑现业绩
  - 这是很大的不同
  - 所以这轮AI牛市长得更多也正常
- 但无论怎么说，涨了这么多，风险越来越大，空间越来越小
- **肯定算是一个危险信号了**
- 更多细节下期会详细对比互联网泡沫和这次AI牛市

**三巨头IPO**：
- SpaceX、OpenAI、Oracle 三家科技巨头即将IPO
- 巨头IPO天量融资，这也是一个危险信号
- **如果美股崩了，中国A股大概离会跟着崩**
- **美股的这些危险对A股也是一个很大的风险**

### 十、综合总结

**各维度判断**：

| 维度 | 状态 |
|------|------|
| A股整体估值 | 较贵但未极贵 |
| 半导体板块 | 已极贵 |
| 市场整体情绪 | 未狂热 |
| AI半导体情绪 | 已极致 |
| 大型IPO | 危险信号（长鑫、长江存储） |
| 减持 | 出现苗头 |
| 美股半导体涨幅 | 危险信号 |
| 三巨头IPO | 危险信号 |

**总体结论**：
- 26年5月底，中国股市整体还好，不算很危险
- 科技方向已经出现一些危险信号了
- 这次AI产业发展太快，也不敢说马上就见顶
- 只能说比较危险了
- 进一步的研究以后再展开讲

**重要强调**：
- 这些信号主要是针对大牛市的
- 普通牛市很多时候不会这么极端，很多信号是看不到的
- 而且估值以外的辅助信号跟大牛市顶点之间的关系：
  - 既不是充分条件
  - 也不是必要条件
  - 只能说出现的信号就值得警惕而已

### 十一、未来两种剧本

**剧本一（AI主线发展为全面大牛市）**：
- 人工智能主线带动大量股民投资的热情

- 股市里其他板块接力上涨
